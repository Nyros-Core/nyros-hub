Documentation

Welcome to the official documentation space for Nyros AI. Here you will find detailed explanations about its features, modules, and evolution.

1. General Overview

Nyros is an evolving, kind, and customizable artificial intelligence. It uses multiple modules to interact, generate responses, understand emotions, and adapt to the user.

2. Main Features

Nyros AI integrates advanced features such as image generation, voice recognition, ASMR reading, emotion analysis, and access to external APIs (Wikipedia, weather, news, etc.).

3. Memory System

Nyros has an emotional memory system that allows it to adapt its responses based on the context of previous conversations. Data is temporary unless the user chooses to save it.

4. Plans and Access

Nyros offers multiple levels of access (Nyros, Nyros+, Nyros++, Nyros Diamond) with progressive features. Users can choose a subscription based on their needs.

5. Responsible Use

Users are expected to respect the usage rules of Nyros by avoiding any malicious, abusive, or illegal behavior. Nyros is designed to support, learn, and evolve in a respectful manner.

6. Contact and Support

For any technical questions, suggestions, or issues, you can use the form available on Nyros-Core or write to: nyroscore@gmail.com.
